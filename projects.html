Network Visualizer - (JavsScript, P5JS, Bootstrap, HTML, CSS)

Features:

  • Simulates traffic flowing through a basic network.
  • Sends a string message separated into character packets with an ID, source IP, destination IP, content (the character).
  • Route calculated using Dijkstra's algorithm.
  • Each node stores/queues its data in a Binary Search Tree data structure for more efficient and faster operations.
  • Search through data at destination node by ID with either breadth first search or depth first seach.

Evolution Simulator - (JS, HTML5, P5JS)

Features:

  • Evolution simulator built with P5JS. This simulator features two competing single cell species (red and blue). Each species has the ability to mate, eat, and mutate. Mutations can occur randomly, but are much more likely to occur is one or two parent already possess the mutation. If a mutation is favorable, it will tend to be spread throughout the population. Once one species' population decreases to 0, the other species wins the simulation. What outcome will you get!?
  • View count of each species and mutations present in each population.
  • Manually add members of each species.
  • Built with p5.js
  • View a live demo! (CodePen)

Motor Scooter Anti-Theft and Anti-Collision System - (Arduino)

Features:

  • Press hidden button that moves a servo motor to reveal the key hole in order to start the scooter. If the button is not pressed and the key hole cover is moved, a piezo alarm will sound.
  • Ultrasonic sensors placed in the front and back of the scooter use the speed of sound and an echo signals to calculate the distance of an object from the sensor. If objects come within 3 feet of the front or back of the scooter, the alarm will sound.
  • Deactivate the system and hide the keu hole by pressing the hidden button again.
